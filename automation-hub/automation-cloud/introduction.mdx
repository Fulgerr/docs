## Key Features
- **Process Identification**
	Share your own ideas/requests for automation through this collaborative process identification tool.

- **Process & Documentation Bank**
	A central repository containing all the key documents related to a process selected as a candidate for automation.
- **Integration with Task Capture**
	Automation Hub is integrated with [UiPath Task Capture](https://docs.uipath.com/task-capture), a process discovery tool that helps you get detailed insights about automation candidates directly from your employees.
