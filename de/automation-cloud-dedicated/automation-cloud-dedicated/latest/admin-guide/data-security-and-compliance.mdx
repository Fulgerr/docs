Die Automation Cloud<sup class="ph sup">TM</sup> Dediziert und die UiPath® Platform als Ganzes verfügen über mehrere Sicherheitsnachweise, die wir freiwillig ausgefüllt haben. Wir durchlaufen kontinuierlich interne und externe Verifizierungen und überwachen unseren Sicherheitsstatus.

Automation Cloud Dedicated verfügt derzeit über die ISO- und SOC 1-Zertifizierungen. Weitere Informationen finden Sie im [Abschnitt Sicherheit auf uipath.com](https://www.uipath.com/legal/trust-and-security/security#content-2).

Zusätzlich zu den Sicherheitsinformationen auf unserer Website kann UiPath Details zu vielen Betriebs- und Sicherheitspraktiken im Rahmen von NDA zur Kundenprüfung oder für die Sicherheitsexperten potenzieller Kunden anbieten. Um Details anzufordern, wenden Sie sich an Ihr Konto-Team von UiPath.
